Diminishing Returns (DR) helps tracking your teammate and own diminishing returns on hostile _players_ by attaching icons to unit frames.

h2. Features

* display running diminishing returns as icons, showing the diminishing status and remaining time,
* uses DR category database from "DRData-1.0":http://wow.curse.com/downloads/wow-addons/details/drdata-1-0.aspx,
* select which categories to show or enable the learning option so they are detected automatically,
* main options and icon layout options are available using @/dimret@ and in the addon tab of interface options,
* test mode available using @/drtest@.
* "LibDualSpec-1.0":http://wow.curse.com/downloads/wow-addons/details/libdualspec-1-0.aspx support for per-spec profiles,
* "AddonLoader":http://wow.curse.com/downloads/wow-addons/details/addon-loader.aspx support to reduce logon time.

h2. Supported unit frames

* Blizzard target, focus and arena enemy frames,
* "Shadowed Unit Frames":http://wow.curse.com/downloads/wow-addons/details/shadowed-unit-frames.aspx: target, focus and arena frames (ShadowedUF_Arenas),
* "Gladius":http://www.wowinterface.com/downloads/info11447-Gladius.html (arena unit frames),
* "X-Perl UnitFrames":http://wow.curse.com/downloads/wow-addons/details/xperl.aspx: target and focus.
* "oUF":http://www.wowinterface.com/downloads/info9994-oUF.html: target and focus.
* "Stellar UF":http://www.wowinterface.com/downloads/fileinfo.php?id=11182: target and focus.
* "oUF_Adirelle":http://wow.curse.com/downloads/wow-addons/details/ouf_adirelle.aspx: target and focus.
* (experimental) Blizzard nameplates, you have to install "LibNameplate-1.0":http://wow.curse.com/downloads/wow-addons/details/libnameplate-1-0.aspx to enable this,
* (untested) "PitBull Unit Frames 4.0":http://wow.curse.com/downloads/wow-addons/details/pitbull4.aspx: target and focus.
* (untested) "ag UnitFrames":http://wow.curse.com/downloads/wow-addons/details/ag_unitframes.aspx: target and focus.

The icon layout for each frame is configurable using the "frame layout" entry in interface options.
